Remotion Components Reference

Imports:

import {
    AbsoluteFill,
    Sequence,
    Img,
    useCurrentFrame,
    interpolate
} from "remotion";

Rules:

1. Use AbsoluteFill as the root component.

2. Use one Sequence per storyboard scene.

3. Use Img for images.

4. Use captions using text elements.

5. FPS = 30.

Frame formula:

frames = duration_in_seconds * 30

Examples:

<Sequence from={0} durationInFrames={120}>

    <Img src="IMG_0704.jpg" />

</Sequence>

Transitions:

fade
crossfade
zoom
slide
cut

Fade Example:

const frame = useCurrentFrame();

const opacity = interpolate(

    frame,

    [0, 30],

    [0, 1]

);
