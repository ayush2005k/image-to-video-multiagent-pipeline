Remotion Components Reference

Use these imports:

import { AbsoluteFill, Sequence, Img } from "remotion";

Rules:

- Use AbsoluteFill as the root component.
- Use one Sequence for each scene.
- Use Img for images.
- Use captions using normal text elements.
- Sequence duration is given in frames.
- FPS = 30.

Example:

<AbsoluteFill>

    <Sequence from={0} durationInFrames={120}>

        <Img src="IMG_0704.jpg" />

    </Sequence>

</AbsoluteFill>

Frame Formula:

frames = duration_in_seconds * 30

Transitions:

fade
crossfade
zoom
slide
cut
